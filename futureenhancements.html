Future Enhancements

Future enhancements will mainly include implementing the full application. As of now, we have developed the "Lite" version of the application which features the reference material. The future application will have:
  • Make new jobs which save information for that job
  • Login information
  • Forgot Password integration
  • Searching for jobs
  • Make new jobs which save information for that job
  • Camera integration for onsite pictures
  • Barcode reader for simplification of data acquisition
  • GPS integration for coordinate entry
